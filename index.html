Elementos en bloque

Los elementos en bloque suelen ocupar todo el ancho disponible del dispositivo (respetando márgenes y paddings) y comienzan en una nueva línea.

Elemento Descripción
<article> Contenedor de contenido independiente.
<aside> Barras laterales.
<blockquote> Citas de texto largas de otra fuente.
<canvas> Área de dibujo para gráficos y animaciones con JavaScript.
<div>* Caja (contenedor) genérico para estructurar contenido.
<dl> Lista de definiciones con términos y descripciones.
<fieldset> Agrupación de campos de un formulario.
<figure> Descrición de una imagen
<footer> Pie de página de un documento o sección.
<form> Contenedor para formularios y entradas de usuario.
<h1> – <h6> Encabezados (títulos) de distintos niveles para estructurar el contenido.
<header> Encabezado de página o sección.
<hr>* Línea horizontal
<li> Elemento dentro de una lista <ol> o <ul>.
<main> Caja con el contenido principal de la página web.
<nav> Menú principal de la página
<ol> Lista ordenada con números o letras.
<p> Párrafo de texto.
<pre>* Texto preformateado que respeta espacios y saltos de línea.
<section> Cajas con contenido temático.
<table> Tabla para organizar información en filas y columnas.
<ul> Lista con viñetas

Elementos en línea

No interrumpen el flujo del contenido, solo ocupan el espacio necesario y pueden estar dentro de una línea de texto.

Elemento Descripción
<a> Crear enlace o marcador.
<abbr> Abreviatura
<b> Negritas
<br> Salto de línea
<button> Botón de formulario.
<cite> Citar fuente o referencia.
<code> Texto con formato de código, como <kbd> y <var>.
<em> Énfasis
<i> Cursiva
<img> Imagen
<input> Campo de entrada de datos en un formulario.
<label> Etiqueta descriptiva para un elemento de formulario.
<mark> Texto resaltado
<q> Cita breve dentro del texto.
<s> Tachado
<select> Lista desplegable
<span> Caja en límea (para seleccionar texto)
<strong> Importante!
<sub> Subíndice
<sup> Superíndice

Cambio de elemento de bloque a línea y viceversa

Por defecto, los elementos en bloque como por ejemplo <div>, <li>, <section> y <article> son de bloque, lo que significa que ocupan todo el ancho disponible y generan un salto de línea.

Si quieres que un elemento en bloque se comporte como en línea, se usa en CSS la regla: 

    elemento {
      display: inline;
    }

En cambio, los elementos como <a>, <span>, <strong>, <em> o <img> son elementos en línea, es decir, no interrumpen el flujo del texto. Si se desea que un elemento en línea actúe como un bloque se usa:

    elemento {
      display: block;
    }

¿Qué es display: inline-block;?

Este valor combina lo mejor de ambos mundos:

  • Se comporta como un elemento en línea, por lo que no genera saltos de línea.
  • Admite propiedades de bloque, como width y height

Ejemplo:

    elemento  {
      display: inline-block;
      width: 150px;
      height: 40px;
      background-color: lightblue;
    }

Propiedades específicas de los elementos en bloque

  1. Ocupan todo el ancho disponible

    Por defecto, un elemento en bloque se expande horizontalmente hasta ocupar el ancho completo de su contenedor padre, incluso si su contenido es menor.

  2. Generan un salto de línea antes y después

    Cada vez que se usa un elemento en bloque, este hace que el siguiente contenido aparezca en una nueva línea, separándolo visualmente.

  3. Se pueden modificar con width y height

    A diferencia de los elementos en línea, los bloques permiten establecer un ancho (width) y una altura (height) sin problemas.

  4. Admiten márgenes (margin) y relleno (padding) en todas direcciones

    Se les puede aplicar márgenes y relleno en todos los lados de un bloque (top, right, bottom, left), sin restricciones.

  5. Pueden contener otros elementos en bloque o en línea

    Un elemento en bloque puede albergar otros bloques o elementos en línea sin problemas. Por ejemplo, <div> puede contener <p>, <h1>, <span>, etc.

  • El html tiene un color verde.

  • El ancho del body es de 80%

  • Para mostrar una imagen como fondo del body, usa la siguiente regla css:
    background-image: url('../img/selva.jpeg');
